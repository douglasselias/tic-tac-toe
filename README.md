# tic-tac-toe

Tic Tac Toe game with [negamax](https://en.wikipedia.org/wiki/Negamax) algorithm for AI

Using C and [Raylib](https://www.raylib.com/index.html)

Build with this command:

`gcc -o main main.c -lraylib && ./main`
